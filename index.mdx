  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get up and running in minutes</p>
  
  <Steps>
    <Step title="Install Dependencies">
      Clone the repository and install the required packages.
      
      ```bash
      npm install
      ```
      
      <Note>The application requires Node.js and npm to be installed on your system.</Note>
    </Step>
    
    <Step title="Configure Firebase">
      Set up your Firebase project and add your credentials to the configuration file.
      
      ```javascript
      // src/js/data/firebase-config.js
      export const firebaseConfig = {
        apiKey: "your-api-key",
        authDomain: "your-auth-domain",
        projectId: "your-project-id",
        storageBucket: "your-storage-bucket",
        messagingSenderId: "your-messaging-sender-id",
        appId: "your-app-id"
      };
      ```
      
      <Tip>Visit the [Firebase Setup Guide](/guides/firebase-setup) for detailed configuration instructions.</Tip>
    </Step>
    
    <Step title="Start the Application">
      Run the development server to see your application in action.
      
      ```bash
      npm run dev
      ```
      
      Your application will be available at `http://localhost:3000`.
    </Step>
  </Steps>
</div>
